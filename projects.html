Built Things

Research tools, experimental languages, and software artifacts. Some are useful. Some are deliberately not. All are finished.

Research Tools

Instruments and utilities
NeuroSync Python

639-line telemetry tool for cross-referencing FlowTime EEG biometric data with Sanding Scale cognitive load reports. tkinter interface with real-time visualization.

biometrics eeg tkinter
Sanding Scale Tracker HTML

Real-time cognitive load monitoring interface. Input current sanding level (0.0–12.0+), view behavioral anchors at each threshold, track session history.

monitoring self-report
MTGPython Python

Learn Python through Magic: The Gathering card mechanics. Translates card text to Python classes. Teaches OOP, conditionals, and state management through a domain the user already masters.

education mtg oop

Experimental Languages

Esoteric programming
FrogPond Custom

A haiku-based programming language. Programs must be written in valid 5-7-5 syllable structure. Compiles patience and poetry into executable logic. Includes REPL.

esolang haiku repl
PleaseScript Python

A language with no whitespace and mandatory manners. All whitespace is stripped. Commands only execute if they begin with "please." Rude instructions are silently ignored.

esolang no-whitespace polite

Wonkys

Experimental software artifacts — deliberately useless, accidentally load-bearing
Entropic Atomic Hostility HTML

Win95-style window containing a live seismograph. Press "Get Value" to receive nine instances of the word "help." Close the window for a custom BSOD. The hostility is permanent.

win95 canvas animation
Win95 Melting Folders HTML

My Computer window where the folder icons progressively melt. Status bar reads: "Just try your best I guess?"

win95 css-animation
SOUL_RADIO HTML

Audio synthesizer with retro interface. Generates sound from configurable waveforms. Version 1.1.

audio web-audio-api
ASCII_CHAOS COBOL

Experimental text artifact written in COBOL. Because some chaos requires a language from 1959.

cobol retro
progresspanic.py Python

A loading bar that panics. At 85% it starts worrying. At 90% it escalates. At 99% it exits and creates a haunted readme filled with variations of "oh no."

cli comedy
PyDeck Cards HTML

Collectible card game template system for learning Python. Cards represent programming concepts with rarity, cost, and combat stats. Bug encounter cards serve as enemies.

education card-design css

Apps & Tools

Things that do things
GardenDeck HTML

Garden tracker that presents your plants as Magic: The Gathering cards. Auto-populates scientific names and growing data via Perenual API. Card flip reveals maintenance logs. PWA-ready.

pwa api mtg-aesthetic
WonkyPad Python

Text editor with Win95 aesthetic. Syntax highlighting, auto-close brackets, and 13 wonky text transforms including Sarcasm Case, uwu, and Zalgo. Because developer tools should respect users.

tkinter editor win95
UselessTracker HTML

An app that tracks itself. Measures whether it's installed (yes), whether it's open (yes), session duration, and button presses. Embedded AI observer provides philosophical commentary via Claude API.

self-referential claude-api csv-export
KairosFileSifter Python

File organization utility. Sifts, sorts, and manages files with a focus on getting things where they belong without fuss.

utility files
Smoosher Python

Text compressor and manipulator. Version 2. Smooshes things together that probably shouldn't be smooshed.

text utility
RockSimQT Python

Rock simulation. Qt interface. Rocks do what rocks do, which is mostly just sit there. Compellingly.

simulation qt
