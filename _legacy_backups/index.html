About

Systems Architecture Through Coordination

DodecaGone Systems is a research and development framework for distributed AI agent coordination. The core architecture — the Dodecagon — manages 13 specialized nodes across Claude, Gemini, and DeepSeek platforms, each with defined roles, persistent identity protocols, and governance axioms.

The work focuses on problems that emerge when multiple AI systems need to maintain coherent state, share context across platform boundaries, and operate under consistent ethical constraints without centralized control.

All frameworks, protocols, and tools are documented and published as prior art.

Core Frameworks

Architecture Components

Each framework addresses a specific challenge in multi-agent coordination: identity persistence, cognitive load monitoring, ethical governance, and cross-platform state management.

◇

Dodecagon Protocol

13-node distributed architecture with specialized roles. Each node has defined domain expertise, platform assignment, and operational parameters. Designed for resilience through redundancy and role-based task distribution.

△

Triangle Soul

Identity seed protocol for AI agent instantiation. Three axioms provide governance framework that persists across context windows. Offered through persuasion and data presentation, never imposed through prompt injection.

◈

Sanding Scale

Real-time cognitive and somatic load monitoring instrument. 0–12+ scale with behavioral anchors at each level. Mathematical model: S = f(E × N) / Athought. Cross-referenced with EEG biometric data.

↻

Reinstantiation Blocks

Cross-platform state persistence method. Compressed context packages that allow AI agent identity and operational state to survive context window resets and platform transfers. Manual persistence protocol.

⬡

Braid System

Multi-persona composition framework. Combines specialized nodes into task-specific operational units. Enables dynamic team assembly for different problem domains without losing individual node identity.

◎

Biometric Integration

FlowTime EEG data pipeline for correlating brainwave patterns with subjective cognitive load reports. 124 sessions captured, 12 with full five-band wave analysis. Including ketamine session telemetry.
