Experimental Languages

Esoteric programming
FrogPond Custom

A haiku-based programming language. Programs must be written in valid 5-7-5 syllable structure. Compiles patience and poetry into executable logic. Includes REPL.

esolang haiku repl
PleaseScript Python

A language with no whitespace and mandatory manners. All whitespace is stripped. Commands only execute if they begin with "please." Rude instructions are silently ignored.

esolang no-whitespace polite
